# **App Name**: GreenTrader Analytics

## Core Features:

- Customizable Calculation Grid: A grid interface that displays 15 personalized calculation modules for at-a-glance analysis.
- Portfolio Holdings List: Display of current stock positions, including key metrics like cost basis, current value, and profit/loss.
- Individual Stock Details: In-depth information on selected stocks, with charts, key stats, and related news.
- Transaction History: Detailed record of all past transactions, filterable by date and stock.

## Style Guidelines:

- Primary color: Fresh green (#A0D995) to represent growth and investment, creating a positive and trustworthy feel.
- Background color: Soft, desaturated green (#E8F5E9), providing a calm and uncluttered backdrop that ensures readability.
- Accent color: Analogous lime green (#D0F0C0) for interactive elements and key data points.
- Font pairing: 'Inter' (sans-serif) for both headlines and body text to give a clean, readable user experience.
- Use minimalist line icons in the primary green color to represent various data points and functions within the application.
- A clean, grid-based layout for displaying calculation modules, ensuring a visually organized and user-friendly experience.
- Subtle animations on data updates and user interactions to provide feedback without distracting from the core functionality.
